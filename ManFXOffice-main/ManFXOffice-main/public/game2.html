Update Report (v6.2)

  • Critical Bugfix (Boss Rush): Fixed bug where leveling up from enemies incorrectly reset the boss timer.
  • Feature (Boss Rush Progression):** Boss Rush now spawns Base, Titan/Fusion, and God bosses on a 5/20 wave cycle, just like Standard mode.
  • Sandbox: Added "Clear All Enemies" button.
  • Sandbox: Fixed "Reset Player" to also reset all stats and upgrades.
  • Gamemodes: Added "Sandbox" and "Boss Rush" modes.
  • Titan/Fusion Bosses: Added `Machine Gun Boss`, `General Boss`, `Slicer Boss`, `Cloner`, `Engineer`, and `Mage`.
  • Final Boss: Added the `GodBoss`.
  • Admin Panel Unlock: Admin Panel now unlocks *per-run* by defeating the `GodBoss`.
  • New Upgrades: Added `Adrenaline`, `Heavy Caliber`, `Lightweight Mags` (Basic) and `Edge Bounce`, `Ricochet`, `+5 Bounces`, `Velocity/Duration x1.5/x3` (Rare).
  • Pause Menu: Added an "Upgrades" tab to view all collected upgrades.
